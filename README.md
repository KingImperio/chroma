# Chroma

HSL-based color palette generator with five harmony modes, live preview, and CSS variables export.

## What it does

Chroma turns a single hex color into a 5-color palette using HSL harmony math. It supports Complementary, Analogous, Triadic, Split-Complementary, and Monochromatic modes. Designers and developers use it to build consistent color systems without guessing at relationships.

## Tech

- HTML / CSS / JavaScript (Vanilla)
- Pure HSL / RGB / WCAG contrast math, no color library
- Satoshi from Fontshare

## Run locally

Download or clone the repo. Open `index.html` in a browser. No build step required.
